🚀 Como conectar o relay 1. Abra o SPX → Entrega → Visão geral dos motoristas
2. Pressione F12 → Console
3. Cole o conteúdo de relay_spx_lpr24.js e Enter
4. Clique Procurar no SPX — dados salvos automaticamente
5. Este painel atualiza a cada 30 segundos via Supabase

⚠ Primeiro uso — crie a tabela no Supabase:
CREATE TABLE IF NOT EXISTS fleet_monitor ( id BIGSERIAL PRIMARY KEY, hub_id TEXT NOT NULL, ts TIMESTAMPTZ DEFAULT NOW(), drivers JSONB, summary JSONB, total INT ); ALTER TABLE fleet_monitor ENABLE ROW LEVEL SECURITY; CREATE POLICY "anon_all" ON fleet_monitor FOR ALL TO anon USING (true) WITH CHECK (true);
